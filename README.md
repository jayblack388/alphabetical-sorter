## Install

Search for `Alphabetical Sorter` in the VS Code/Cursor Extensions panel and install it.

Or install from [Open VSX](https://open-vsx.org/extension/jayblack388/alphabetical-sorter).

## Usage

1. Select the text you want to sort
2. Use keyboard shortcut: `Cmd+Shift+A` (Mac) / `Ctrl+Shift+A` (Windows/Linux)
3. Or open Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`) and select `Alphabetical Sorter`
